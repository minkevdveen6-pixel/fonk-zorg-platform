# FONK

**wêr feroaring begjint**

Prototype van een warm, mensgericht veranderplatform voor zorgorganisaties.

## Wat zit erin

- Next.js App Router
- TypeScript
- Tailwind CSS
- Warme, zorggerichte stijl met zachte kleuren en Friese accenten
- Projectleidersoverzicht met voorbeeldacties
- Planhulp met voorbeeldoutput
- Directie/MT-prioritering met maximaal 3 PRIO-projecten
- Organisatiebrede projectkaart met klikbare projecten
- Medewerkersomgeving en ideeënhub
- FONK Scan
- Regionale samenwerking en impactoverzicht als toekomstmodule
- Mockdata zonder backend of database

## Lokaal starten

```bash
npm install
npm run dev
```

Open daarna:

```text
http://localhost:3000
```

## Build controleren

```bash
npm run build
```

## Belangrijke mappen

```text
app/                 Routes en metadata
components/          Herbruikbare UI-componenten
lib/fonk-data.ts     Projecten, ideeën en afdelingenadvies als voorbeelddata
lib/scan.ts          FONK Scan scorelogica
```
